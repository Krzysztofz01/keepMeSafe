KeepMeSafe

Are you worried about the security of your online accounts? With this application you can check the security of your passwords. Check if it has the right number and combinations of characters and thanks to the external API, you can check if your password has ever leaked from any database. If necessary, you can generate a new, secure password that meets all the requirements, and stafy safe.

According to Kaspersky's password tester, KeepMeSafe-generated password cracks would take 10,000+ centuries! That's a huge amount of time ;)

KeepMeSafe is using SecList password list to check for leaked password.
